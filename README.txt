========================================
  桌面时钟 (Desktop Clock) v1.0
========================================

[简介]
  一个无边框、半透明、始终置顶的桌面时钟。
  基于 Python 标准库 tkinter，无需安装第三方依赖。

[依赖]
  Python 3.x（3.10+ 推荐）
  无需额外依赖，tkinter 随 Python 一起安装。

[运行方式]
  方式一（推荐 — 无控制台窗口）：
    cd D:\DesktopClock
    pythonw main.py

  方式二（会弹出控制台窗口）：
    cd D:\DesktopClock
    python  main.py

[功能说明]
  - 实时显示当前时间 HH:MM:SS，每秒刷新
  - 显示当前日期（年月日 + 星期）
  - 无边框、置顶显示
  - 半透明深色背景（透明度约 82%），白色文字
  - 按住窗口任意位置可拖拽移动
  - 右键弹出菜单，提供"退出"选项
  - 默认位于屏幕右下角

[自定义]
  如需调整外观，编辑 main.py 顶部配置常量：
    BG_COLOR    — 背景颜色（十六进制）
    FG_COLOR    — 文字颜色
    TIME_FONT   — 时间字体
    DATE_FONT   — 日期字体
    WIN_ALPHA   — 窗口透明度（0.0~1.0）
    WIN_MARGIN  — 距屏幕边缘距离（像素）
